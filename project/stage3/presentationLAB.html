Цель работы  
Использование Hydra для подбора или взлома имени пользователя и пароля.  
Выполнение лабораторной работы  
1. Скачиваем текстовый документ rockyou.txt.tar.gz с паролями для Linux командой sudo gzip -d  
rockyou.txt.tar.gz.(рис.[1])  
1
2. Установливаем в браузере расширение для просмотра cookie и копируем значение PHPSESSID для  
дальнейшей работы.(рис.[2])  
2
3. Запускаем работу Hydra. Для авторизации используется html форма, которая отправляет методом POST  
запрос вида username=root&password=test_password. Выбираем любую выданную пару логина и пароля.  
(рис.[3])  
3
4. Заходим обратно на сайт и вводим выбранную пару логин-пароль и получаем результат взлома.(рис.[4])  
4
Вывод
